# VOoM
forked from vim-voom/VOoM

## Installation
```bash
Plug 'zhenyangze/VOoM'
let g:voom_ft_modes = {'markdown': 'markdown', 'c': 'fmr2', 'cpp': 'fmr2', 'markdown.mkd': 'markdown', 'tex': 'latex'}
```

## Quick Start
```bash
:Voom [markup]
:Voom [filetype]
```
